Git-入门指南

(前提安装git，注册好自己的github) 1.cd到本地根目录，让本地成为仓库路

git init

2.将本地项目文件添加到仓库(实际上是提交到暂存区)

git add .

3.将暂存区文件提交到仓库

git commit -m "注释语句"

4.将本地仓库关联到github（前提你github上有一个仓库Repository）

git remote add origin https://github.com/YTU94/your-repository-name.git

5.上传之前先pull一下(具体用来同步本地仓库的差异)

git pull origin master

6.上传到github

git push -u origin master

常用命令功能

查看仓库当前的状态

git status

查看文件做了哪些修改

git diff text.js

查看从最近到最远的提交日志

git log

版本回滚

git reset --hard HEAD^

记录你的每一次命令

git reflog

批量删除分支

git branch |grep 'branchName' |xargs git branch -D
